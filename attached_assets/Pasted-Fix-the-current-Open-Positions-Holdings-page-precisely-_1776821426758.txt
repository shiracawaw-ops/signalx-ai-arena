REQUIRED POSITION CLASSIFICATION
==================================================

Implement a strict classification layer for every asset shown on this page.

Each asset must be classified into exactly one of these categories:

1) Active Position
Definition:
- there is a real tracked open trading position
- quantity is meaningfully open
- quantity is above exchange sellable minimum
- position is still part of the live trade lifecycle

2) Residual Position
Definition:
- a trade was closed or mostly closed
- a meaningful remainder still exists
- it may still be sellable depending on exchange minimums
- it should be shown separately from fully active positions

3) Dust / Unsellable Remainder
Definition:
- owned quantity exists
- but its current notional value is below exchange minimum sellable amount
- or quantity is too small to close legally
- this must not be treated like a normal active position

4) Wallet Holding Only
Definition:
- asset exists in the wallet
- but is not tied to an active tracked trading position
- do not present it as an active open position
